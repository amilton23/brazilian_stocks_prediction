# Project of Brazilian Stocks Prediction
---

## Overview

This project aims to predict the stock prices of Brazilian companies using machine learning techniques. The project uses historical stock price data and other relevant features to train a model that can predict future stock prices.

## Development plan

## Data Sources

The project uses the following data sources:

- **Stock Price Data**: Historical stock price data from Yahoo Finance
- **Economic Indicators**: Economic indicators from the Brazilian Central Bank
- **Company Financials**: Company financials from the Brazilian Securities and Exchange Commission (CVM)
